# K72.ca Agency Clone

This is a **clone of the K72.ca agency website**, built using **React** for the frontend and **GSAP** for smooth animations. It showcases modern web development techniques including responsive design, interactive animations, and a single-page application structure.

## Features

- Fully responsive layout across desktop, tablet, and mobile  
- Smooth scrolling animations with **GSAP**  
- Interactive elements (hover effects, scroll-triggered animations)  
- Component-based architecture using **React**  
- Reusable navigation and sections for easy scalability  

---

## Technologies Used

- **React** - Frontend library for building UI components  
- **GSAP (GreenSock Animation Platform)** - For animations and scroll-triggered effects  
- **React Router DOM** - For page routing (if applicable)  
- **Tailwind** - Styling and responsive design
